Politica de Privacidad de EASY SATJE

Ultima actualizacion: 26 de mayo de 2026

EASY SATJE es una extension de Chrome destinada a ayudar a abogados y usuarios del sistema SATJE a capturar informacion visible de expedientes judiciales y generar borradores de escritos.

Informacion que procesa la extension

La extension puede procesar informacion visible en la pagina activa del sistema SATJE cuando el usuario decide usar la herramienta. Esta informacion puede incluir numero de proceso, judicatura, tipo de accion, actor, demandado, actuaciones visibles y texto capturado del expediente.

Uso de la informacion

La informacion procesada se utiliza exclusivamente para mostrar datos del expediente al usuario y generar borradores de escritos en formatos descargables como DOCX y PDF.

Almacenamiento local

EASY SATJE puede guardar localmente preferencias de uso y datos configurados por el usuario, como informacion del abogado, opciones de la extension y sesion local. Esta informacion se conserva para facilitar el uso de la herramienta y evitar que el usuario tenga que ingresarla nuevamente.

Comunicacion con el backend

Cuando el usuario solicita generar un borrador, la extension puede enviar el texto capturado y los datos necesarios del expediente al backend de EASY SATJE para procesar la solicitud. La extension no utiliza estos datos para publicidad, rastreo de navegacion ni venta a terceros.

Permisos de Chrome

La extension utiliza permisos como activeTab, scripting, storage, sidePanel y downloads para funcionar correctamente. Estos permisos se usan para leer la pagina activa autorizada por el usuario, mostrar la interfaz de trabajo, guardar preferencias locales y descargar documentos generados.

Codigo remoto

EASY SATJE no descarga ni ejecuta codigo remoto dentro del navegador del usuario. El codigo de la extension se incluye en el paquete publicado en Chrome Web Store.

Comparticion de informacion

EASY SATJE no vende, alquila ni comparte informacion personal del usuario con fines comerciales. La informacion se utiliza solo para prestar las funciones propias de la extension.

Seguridad

Se aplican medidas razonables para proteger la informacion procesada por la herramienta. Sin embargo, el usuario debe revisar los documentos generados antes de utilizarlos o presentarlos dentro de cualquier procedimiento judicial.

Contacto

Para consultas relacionadas con esta politica de privacidad o el tratamiento de informacion por EASY SATJE, el usuario puede contactar al responsable del proyecto a traves de los canales oficiales de soporte indicados en la publicacion de la extension.
